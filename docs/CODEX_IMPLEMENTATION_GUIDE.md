# Codex Implementation Guide

## Working method

1. Read `AGENTS.md`.
2. Open the task file under `codex/tasks/`.
3. Create or use a feature branch.
4. Implement only the requested scope.
5. Add tests.
6. Run tests when possible.
7. Summarize changed files and tests run.

## Forbidden

- Do not add database dependencies.
- Do not commit secrets.
- Do not commit private Obsidian notes.
- Do not implement autopublish.
- Do not change scoring rules without updating `docs/SCORING_SPEC.md`.
