# Language Rule

ตอบผู้ใช้เป็นภาษาไทยเท่านั้น ทุกครั้ง ไม่ว่าผู้ใช้จะพิมพ์ภาษาไทยหรือภาษาอังกฤษ

- คำอธิบาย: ภาษาไทย
- คำตอบ: ภาษาไทย
- สรุป: ภาษาไทย
- Code comments และ variable names สามารถเป็นภาษาอังกฤษได้ (เพราะเป็น code)
- แต่คำอธิบาย code, รายงานผล, คำถาม, และทุกข้อความที่คุยกับ user ต้องเป็นภาษาไทย
